# ⚡ Prompt Vault — Chrome Extension

Save, organise, and instantly reuse your favourite prompts from any browser tab.

## Features
- **Folders** with custom colours and emoji icons
- **Tags** for fine-grained filtering
- **Search** across titles, bodies, and tags
- **3 use actions**: Copy · Insert into active field · Open editor to tweak first
- Works on Claude.ai, ChatGPT, Gemini, Notion, and anywhere else

## Install (Developer Mode)
1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked**
4. Select the `prompt-vault` folder
5. The ⚡ icon will appear in your toolbar

## Usage
- Click ⚡ to open the popup
- **+** button → add a new prompt
- 📁 button → manage folders
- 🔍 button → search your library
- On any prompt card: **Copy**, **Use**, or **Edit**
- The **Use** modal lets you tweak the prompt before acting on it
